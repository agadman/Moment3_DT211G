CSS preprocessorer

Den här webbplatsen har jag byggt som ett delmoment i kursen Frontendbaserad webbutveckling. Momentet handlar om CSS-preprocessorer.

Effektivitet - Med SASS går det snabbare och smidigare att skriva CSS tack vare variabler, mixins och nesting, vilket sparar tid och minskar upprepning.

Struktur - Med partials och moduler kan koden delas upp i mindre filer, vilket gör den mer organiserad och lättare att arbeta med i stora projekt.

Återanvändbarhet - Mixins och variabler gör det lätt att återanvända kod, vilket gör uppdateringar enklare och minskar risken för att designen blir inkonsekvent.
